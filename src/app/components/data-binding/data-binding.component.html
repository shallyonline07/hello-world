===Data Binding ====

1. One Way Binding
  • Component to View
    • Interpolation
    • Property Binding: Bind component property to HTML element property
    • Attribute Binding: Sometimes there is no HTML element property to bind to, so we can use [attr.propertyName]. for ex: [att.colspan]
  • View to Component
    • Event Binding: (click)
2. Two Way Binding: when it performs property binding+ event binding. property binding is done first then event binding[(ngModel)]... We need to import FormsModule from '@angular/forms' in app.module.ts file.
