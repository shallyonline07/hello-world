===== Types of decorators =====

  Class decorators: e.g. @Component and @NgModule, @Injectable, @Directive, @Pipe
  Property decorators: For properties inside classes, e.g. @Input, @Output, @ViewChild, @ViewChildren
  Method decorators:  For methods inside classes, e.g. @HostListener
  Parameter decorators: For parameters inside class constructors, e.g. @Inject, @Optional
